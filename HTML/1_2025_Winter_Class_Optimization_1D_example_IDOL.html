▣ 최적화란?¶

  • 설계 영역내에서 제한조건을 만족시키면서 목적함수를 최소화 (or 최대화)하는 과정
  • 최적화 알고리즘에 의해 설계 변수가 반복적으로 업데이트되어 최적해가 구해짐
  • 다양한 최적화 알고리즘들이 존재 (Gradient-based optimization, heuristic optimization, surrogate-based optimization)

※ 예제 문제 (Unconstrained optimization)¶

  • 설계 공간: 0~1
  • 목적 함수: obj = (6x - 2)2 sin(12x - 4)를 최소화
  • 제한 조건: 없음
  • 다양한 초기값과 최적화 알고리즘을 적용하여 최적화 결과 비교해보기
  • True 최적해: xopt = 0.75724875, objopt = -6.02074006

def obj(x): # 목적함수 (함수 이름은 자유롭게 설정 가능)
    return ((6*x-2)**2)*np.sin((6*x-2)*2) # x를 입력했을때 나오는 함수 값
